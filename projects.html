Java project

Object-Oriented Java Program calculates the volume of donut or torus

Description:

The program is designed to calculate the volume of donut or torus. The process involves understand the problem, design and describe the algorithm using UML class diagram, pseudocode, testing the workflow of the algorithm using test tables, converting the algorithm to java code. Final steps involve testing and debugging the program

The java program consists of two classes DonutLauncher and Donuta. Getter, Setter, constructor methods, are well utilized in this project. The figure below shows the results of volume calculation using Eclipse IDE

Database Project

Database Design for XYZ manufacturing company

Description:

The project represents the database design for XYZ manufacturing company automated application system. The database is designed to meet the company requirement of new application which allows decision support. By converting existing index file system to relational database management system. The new database design helps to provide customer details, Order details, discount information, product reports, product inventory data and materials required for the manufacturing company.
